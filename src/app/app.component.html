What is the Hustle N’ Code Hackathon?

TXT’s Hustle N’ Code Hackathon is an annual Winter event aimed at inspiring youth who live in public housing developments, inspiring youth that live in Nickerson Gardens to address pressing issues in their communities using technology. Our hackathons are unique by bringing tech companies into historically marginalized communities like South LA. For nine hours, students learn to code and develop mobile apps with the help of more than 150 tech industry professionals.

Why Should YOU be part of the Hustle N' Code Hackathon?

  • Free One Day Coding Event for Students 7th to 12th Grade.
  • Students will learn to code HTML, CSS and JavaScript with Tech Professionals, including Salesforce, Snapchat and Google.
  • Students will work in a team of 4 to address and solve issues ailing their communities through Tech.
  • Students will be exposed to a Tech Start Up Environment.
  • Students will have the opportunity to network with professionals in the Tech Field.
  • By the end of the day Students will have the chance to win prizes like laptops, shirts, and headphones.
